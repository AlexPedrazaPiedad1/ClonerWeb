# ClonerWeb
Este repositorio consiste en el desarrollo de una clonación web centrada en replicar, con la mayor fidelidad posible, el diseño y la experiencia de usuario de páginas reconocidas como la web de McDonald's y la página oficial de Sallenet de La Salle Girona.

El objetivo principal del proyecto es poner en práctica habilidades de desarrollo frontend, prestando especial atención a la estructura, el estilo visual y la interactividad, así como experimentar con la personalización y mejora de los elementos originales.
